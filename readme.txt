== Description ==

Your host doesn't work well with the Matomo WordPress plugin, like, for example, Ionos? Or you host your local Matomo service? 
This is a simplifed plugin you can use for Matomo integration into WordPress. Just install it, set server url and site id and you can go for it ;-)
In current version, campaign keyword support is not included.
